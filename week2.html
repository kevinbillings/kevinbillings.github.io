Ch2-4: Eloquent JavaScript

JavaScript has six datatypes: Strings, Numbers, Booleans, Symbols, Undefined, and NULL

let cookies = 10; // Local

var cookies = 10; // Global

let is available only inside the scope of where a variable is declared - for example a FOR loop. Where var is globally available.

The main difference was that variables declared using let and const have block scope. They also prevent you from overwriting any built-in methods by assignment, whereas using var doesn't.
