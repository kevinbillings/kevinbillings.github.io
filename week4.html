OOP

Object-Oriented Programming (OOP) separates the code into objects and their methods.

Encapsulation - All the processing that needs to be done for a specific task is done internally while the user only sees the result.

Polymorphism - Where the same process can be done for different objects. i.e. an on/off switch or in coding terms sharing the same method.

Inheritance - Takes existing features from an object and adds a few more.

Getters and Setters to safely access the internal variables.

DOM Manipulation

The Document Object Model (DOM) allows programs to dynamically access and update content.

Keeping the code loosely coupled means that it is easy to change/upgrade pieces of code as necessary.
