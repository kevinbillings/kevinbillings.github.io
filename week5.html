Debugging

The console has some really neat built-in functionality for debugging.
  debugger();

Simply entering debugger(); into the code will set break points to check what is currently going on in the code.

Error Objects

Yup, you read that right. We can create error objects for certain exceptions in the code.

const error = new Error('Oops, something went wrong');

Try - Catch - Finally

function imaginarySquareRoot(number) {
 'use strict';
 let answer;
 try {
  answer = String(squareRoot(number));
 } catch(error) {
 answer = squareRoot(-number)+"i";
 } finally {
  return `+ or - ${answer}`;
 }
}
